# Top 20 Real Tasks

## Recommendation

Use these 20 school-focused tasks to shape the first prototype and evaluation set.

Confidence: 90%

Evidence:
- The tasks map to the first product wedge: explain, translate, summarize, and quiz.
- They focus Sprint 1 on curriculum-grounded explanation instead of a broad everyday assistant.
- They are easy to test with pasted text and short questions in a web prototype.

Main uncertainty:
- Real users may rank WhatsApp voice-note tasks higher than text tasks.

What would change this:
- If early testers strongly prefer voice, move voice input into the second prototype cycle.

## Study And Learning

1. Explain a school concept in simple Ghanaian English.
2. Explain a school concept in Twi.
3. Explain a textbook paragraph in simple English.
4. Translate key school vocabulary between English and Twi.
5. Give a Ghanaian everyday example for an abstract concept.
6. Turn class notes into a five-bullet summary.
7. Generate likely BECE-style questions from a topic.
8. Explain why a student's answer is wrong.
9. Solve a math problem step by step.
10. Define science terms with examples from Ghanaian daily life.
11. Compare two concepts, such as evaporation and condensation.
12. Create short revision notes before a test.
13. Generate a quiz and mark the learner's answers.
14. Explain a diagram that the learner describes in words.
15. Help a teacher simplify a lesson introduction.
16. Convert teacher notes into student-friendly language.
17. Translate parent-facing school messages.
18. Explain social studies or civic concepts using Ghanaian examples.
19. Produce vocabulary lists for a topic.
20. Ask follow-up questions to diagnose what the learner does not understand.

## MVP Task Priority

Must support in first prototype:
- Tasks 1, 2, 3, 5, 6, 7, 11, 13, 15, 20.

Should support if time permits:
- Tasks 4, 8, 10, 12, 16, 18, 19.

Defer until multimodal support is stronger:
- Task 14 with actual image upload. Text-described diagrams are acceptable in the first prototype.
